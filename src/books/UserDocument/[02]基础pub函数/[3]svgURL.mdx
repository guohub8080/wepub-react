# {title}

`svgURL` 是一个简单的工具函数，用于将 URL 字符串转换为 CSS `url()` 函数格式。

## 函数签名

```typescript
function svgURL(url: string): string
```

## 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `url` | `string` | 图片或资源的 URL 地址 |

## 返回值

返回 CSS `url()` 格式的字符串，例如：`url("path/to/image.svg")`

## 使用示例

### 基础用法

```typescript
import svgURL from '@pub-utils/common/svgURL';

const bgUrl = svgURL('/images/pattern.svg');
console.log(bgUrl); // url("/images/pattern.svg")

// 在样式中使用
element.style.backgroundImage = bgUrl;
```

### 在 SVG 中使用

```typescript
import svgURL from '@pub-utils/common/svgURL';

const MyComponent = ({ imageUrl }) => {
    return (
        <svg>
            <defs>
                <pattern
                    id="pattern"
                    width="100" height="100"
                    patternUnits="userSpaceOnUse"
                >
                    <image
                        href={imageUrl}
                        width="100" height="100"
                    />
                </pattern>
            </defs>
            <rect
                width="200" height="200"
                fill={svgURL('#pattern')}
            />
        </svg>
    );
};
```

### 与 CSS-in-JS 配合

```typescript
import svgURL from '@pub-utils/common/svgURL';

const styles = {
    backgroundImage: svgURL('/assets/bg.svg'),
    maskImage: svgURL('/assets/mask.svg'),
    listStyleImage: svgURL('/assets/bullet.svg')
};
```

## 应用场景

- **CSS 动态样式** - 动态设置背景图片、遮罩图片等
- **SVG 引用** - 在 SVG 中引用 pattern 或 filter
- **样式对象构建** - 与 CSS-in-JS 库配合使用

## 注意事项

⚠️ **简单封装**
- 该函数只是简单的字符串拼接，不做任何验证
- 确保传入的 URL 格式正确

⚠️ **引号处理**
- 函数会自动添加双引号包裹 URL
- 不需要手动添加引号
