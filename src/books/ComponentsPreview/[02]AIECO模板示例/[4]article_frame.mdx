`ArticleFrame` 是 AIECO 主题的文章内容框架组件，用于包裹文章主要内容区域，并提供底部信息展示功能。

## 导入

```tsx
import ArticleFrame from "@aieco/FullArticleFrame";
```

## 主要功能

| 功能 | 说明 |
|------|------|
| 内容包裹 | 提供统一的容器包裹文章内容 |
| 背景颜色 | 默认浅蓝色背景，支持红色主题 |
| 边框样式 | 灰色边框，底部无边框与底部卡片衔接 |
| 圆角处理 | 上方圆角，底部直角与底部卡片衔接 |
| 底部信息 | 显示文章来源、GitHub、作者等信息 |
| 主题切换 | 支持 `red` 主题，切换配色方案 |

## 内部实现

ArticleFrame 由内容区域和底部区域两部分组成：

```tsx
// 内容区域
<section style={{
  paddingTop: 25,
  paddingBottom: 25,
  borderRadius: 8,
  borderBottomRightRadius: 0,
  borderBottomLeftRadius: 0,
  marginBottom: -25,
  backgroundColor: wechatColors.lightBgBlue,
  border: "1px #d1d5db solid",
  borderBottom: "1px 背景色 solid",
}}>
  {props.children}
</section>

// 底部区域（ArticleBottom 组件）
<ArticleBottom
  borderRadius={8}
  borderColor="#d1d5db"
  lightColor={wechatColors.aiecoLightBlue}
  darkColor={wechatColors.aiecoDarkBlue}
  titleUnderColor="#bfdbfe"
  underTitle={props.underTitle}
/>
```

## Props 参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| `children` | `ReactNode` | 是 | - | 文章内容，通常包含 P、B、ImgFrame1 等组件 |
| `underTitle` | `UnderTitleItem[]` | 是 | - | 底部信息数组，每个元素包含 title 和 name |
| `borderRadius` | `number` | 否 | 8 | 容器圆角大小（px） |
| `articleBgColor` | `string` | 否 | `wechatColors.lightBgBlue` | 文章内容背景色 |
| `theme` | `"red"` | 否 | - | 主题类型，传 `"red"` 切换为红色主题 |

### UnderTitleItem 类型

```tsx
interface UnderTitleItem {
  title: string;           // 标签名称，如"组件来源"、"GitHub"
  name?: string | string[]; // 内容，支持字符串或字符串数组
}
```

## TypeScript 类型定义

```tsx
interface UnderTitleItem {
  title: string;
  name?: string | string[];
}

interface ArticleFrameProps {
  children?: ReactNode;
  borderRadius?: number;
  articleBgColor?: string;
  theme?: "red" | undefined;
  underTitle: UnderTitleItem[];
}
```

## 使用示例

### 基础用法

```tsx
import AIECO from "@aieco/frames/MainFrame";
import HeadAIECO from "@aieco/heads/HeadAIECO";
import ArticleFrame from "@aieco/FullArticleFrame";
import P from "@aieco/docs/P";

const underTitle = [
  { title: "组件来源", name: "WePubReact" },
  { title: "GitHub", name: "https://github.com/guohub8080/wepub-react" },
  { title: "作者", name: "guohub8080" },
];

export default () => {
  return (
    <AIECO>
      <HeadAIECO />
      <ArticleFrame underTitle={underTitle}>
        <P>这是文章内容...</P>
      </ArticleFrame>
    </AIECO>
  );
};
```

### 使用红色主题

```tsx
<ArticleFrame
  underTitle={underTitle}
  theme="red"
  borderRadius={12}
>
  <P>这是红色主题的文章内容...</P>
</ArticleFrame>
```

### 自定义背景色

```tsx
<ArticleFrame
  underTitle={underTitle}
  articleBgColor="#f9fafb"
  borderRadius={10}
>
  <P>这是自定义背景色的文章内容...</P>
</ArticleFrame>
```

## 主题配色

### 默认主题（蓝色）

| 元素 | 颜色 | 说明 |
|------|------|------|
| 内容背景 | `wechatColors.lightBgBlue` | 浅蓝色背景 |
| 装饰条浅色 | `wechatColors.aiecoLightBlue` | 浅蓝色装饰 |
| 装饰条深色 | `wechatColors.aiecoDarkBlue` | 深蓝色装饰 |
| 标题下划线 | `#bfdbfe` | 浅蓝色 |

### 红色主题

| 元素 | 颜色 | 说明 |
|------|------|------|
| 内容背景 | `wechatColors.lightBgYellow` | 浅黄色背景 |
| 装饰条浅色 | `#fbbf24` | 琥珀色装饰 |
| 装饰条深色 | `wechatColors.titleRed` | 红色装饰 |
| 标题下划线 | `#fde68a` | 浅黄色 |

## 样式特点

### 内容区域样式

| 属性 | 值 | 说明 |
|------|------|------|
| 内边距 | 上下 25px | - |
| 圆角 | 8px（可自定义） | 上方圆角 |
| 底部圆角 | 0 | 与底部卡片衔接 |
| 下边距 | -25px | 负边距实现重叠 |
| 边框 | 1px 灰色实线 | 底部无边框 |
| 背景色 | 浅蓝色（默认） | 支持自定义 |

### 底部区域样式

| 属性 | 值 | 说明 |
|------|------|------|
| 圆角 | 8px（可自定义） | 下方圆角 |
| 顶部圆角 | 0 | 与内容区域衔接 |
| 装饰条 | 深浅双色 | 左侧视觉装饰 |
| 标题下划线 | 浅色 | 标签下方装饰线 |

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 底部信息必填 | `underTitle` 是必填参数，至少应包含一个元素 |
| 负边距衔接 | 内容区域使用 `marginBottom: -25` 与底部区域重叠，实现无缝衔接 |
| 圆角处理 | 内容区域底部圆角为 0，底部区域顶部圆角为 0，确保衔接平滑 |
| 主题切换 | 使用 `theme="red"` 切换红色主题，会改变整体配色方案 |
| 背景色覆盖 | 自定义 `articleBgColor` 会覆盖主题背景色 |
