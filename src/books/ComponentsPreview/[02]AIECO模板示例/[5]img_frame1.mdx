`ImgFrame1` 是 AIECO 主题的图片展示组件，支持图片描述文字，带圆角和阴影效果。

## 导入

```tsx
import ImgFrame1 from "@aieco/frames/ImgFrame1";
```

## 主要功能

| 功能 | 说明 |
|------|------|
| 图片展示 | 显示图片，支持自适应宽度 |
| 图片描述 | 支持单行或多行描述文字 |
| 圆角效果 | 默认 8px 圆角，可自定义 |
| 阴影效果 | 默认浅灰色阴影，可自定义 |
| 最大宽度 | 默认 550px，可自定义 |
| 占位图 | 无图片时使用占位图 |

## 内部实现

ImgFrame1 由图片容器和描述框两部分组成：

```tsx
// 图片容器
<section ref={imgRef} style={{
  borderRadius: 8,
  overflow: "hidden",
  backgroundColor: "white",
  boxShadow: "0 0 15px #efefef",
  margin: "0 auto"
}}>
  {/* 图片包裹层 */}
  <section style={{
    width: "100%",
    display: "flex",
    justifyContent: "center",
    alignItems: "center",
  }}>
    <img
      src={url}
      style={{
        width: "100%",
        objectFit: "contain",
        borderTopRightRadius: 8,
        borderTopLeftRadius: 8,
        borderBottomLeftRadius: 有描述 ? 0 : 8,
        borderBottomRightRadius: 有描述 ? 0 : 8,
      }}
    />
  </section>

  {/* 描述框 */}
  <UnderImg description={description} color={color} />
</section>
```

## Props 参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| `url` | `string` | 否 | 占位图 | 图片地址 |
| `description` | `string \| string[]` | 否 | - | 图片描述，支持字符串或字符串数组 |
| `color` | `string` | 否 | 继承 | 描述文字颜色 |
| `maxWidth` | `string \| number` | 否 | `"550px"` | 图片最大宽度，≤0 表示不限制（100%） |
| `borderRadius` | `number \| string \| number[]` | 否 | 8 | 圆角大小 |
| `boxShadow` | `string` | 否 | `"0 0 15px #efefef"` | 阴影效果 |

### borderRadius 详细说明

| 类型 | 示例 | 说明 |
|------|------|------|
| `number` | `8` | 四个角统一圆角 |
| `string` | `"12px"` | 四个角统一圆角（字符串） |
| `number[]` | `[8, 8, 0, 0]` | 分别设置左上、右上、右下、左下圆角 |

## TypeScript 类型定义

```tsx
interface ImgFrame1Props {
  url?: string;
  description?: string | string[];
  color?: string;
  maxWidth?: string | number;
  borderRadius?: number | string | [number, number, number, number];
  boxShadow?: string;
}
```

## 使用示例

### 基础用法（无描述）

```tsx
import ImgFrame1 from "@aieco/frames/ImgFrame1";

<ImgFrame1 url={imgUrl} />
```

### 带描述文字

```tsx
<ImgFrame1
  url={imgUrl}
  description="这是一张测试图片，用于验证图片组件的显示效果。"
/>
```

### 多行描述

```tsx
<ImgFrame1
  url={imgUrl}
  description={[
    "这是第一行描述文字。",
    "这是第二行描述文字。",
    "这是第三行描述文字。"
  ]}
/>
```

### 自定义样式

```tsx
<ImgFrame1
  url={imgUrl}
  description="自定义圆角和阴影"
  borderRadius={12}
  boxShadow="0 4px 20px rgba(0, 0, 0, 0.1)"
  maxWidth="600px"
/>
```

### 不限制宽度（100%）

```tsx
<ImgFrame1
  url={imgUrl}
  description="不限制宽度，图片会占据容器全部宽度"
  maxWidth={0}  // 或 -1，表示不限制
/>
```

### 分别设置四个角圆角

```tsx
<ImgFrame1
  url={imgUrl}
  description="上方圆角，下方直角"
  borderRadius={[12, 12, 0, 0]}  // 左上、右上、右下、左下
/>
```

## 样式特点

### 图片容器样式

| 属性 | 值 | 说明 |
|------|------|------|
| 圆角 | 8px（可自定义） | - |
| 溢出 | `overflow: hidden` | 裁剪圆角 |
| 背景色 | 白色 | - |
| 阴影 | `0 0 15px #efefef` | 可自定义 |
| 对齐 | 居中 | `margin: 0 auto` |

### 图片样式

| 属性 | 值 | 说明 |
|------|------|------|
| 宽度 | 100% | 自适应容器 |
| 最大宽度 | 550px（默认） | 可自定义 |
| 填充方式 | `object-fit: contain` | 保持比例 |
| 顶部圆角 | 继承容器圆角 | - |
| 底部圆角 | 有描述时为 0 | 与描述框衔接 |

### 描述框样式

| 属性 | 值 | 说明 |
|------|------|------|
| 背景色 | `#f3f4f6`（浅灰） | - |
| 内边距 | 上下 8px，左右 12px | - |
| 字体大小 | 14px | - |
| 行高 | 1.6 | - |
| 颜色 | 继承或自定义 | - |
| 顶部圆角 | 0 | 与图片衔接 |
| 底部圆角 | 继承容器圆角 | - |

## 注意事项

| 注意事项 | 说明 |
|----------|------|
| 占位图 | 未提供 `url` 时使用 `placehold.co` 占位服务 |
| 圆角衔接 | 有描述时，图片底部圆角为 0，与描述框无缝衔接 |
| 图片填充 | 使用 `object-fit: contain` 保持图片原始比例 |
| 阴影兼容 | 阴影添加在容器上，避免影响图片显示 |
| 描述颜色 | 未设置 `color` 时使用继承颜色 |
| maxWidth 优先级 | 使用 `!important` 强制应用，避免被覆盖 |
